# Pyscript Unit Converter
__Link:__ https://pyscript-unit-converter.netlify.app/

A simple unit converter made in HTML, CSS and Python3 (Using [pyscript](https://pyscript.net/)) <br>
The code could probably be more efficient and optimised but its just for fun 🤷‍♂️ <br> <br>

__Note:__ If you want to run the code yourself you will need to start a local server (You can't just double-click the HTML file). <br>
For VS Code I recommend using the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension.
